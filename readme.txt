== Description ==

Display Shortcodes for Gravity Form Admin column.

== Installation ==

From the admin panel, Go to your WordPress Admin -> Plugins -> Add New. Search for Shortcodes for Gravity Form. Install and Activate.

From directories, Upload `Shortcodes for Gravity Form` to the `/wp-content/plugins/` directory and activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to use Shortcodes for Gravity Form =

Required Gravity Forms plugins

After installing Shortcodes for Gravity Form you need to go to Gravity Forms section where you can see the list of Shortcodes column .
